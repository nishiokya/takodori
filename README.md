# 🐙 二タコ採り — TAKO TORI

海の生きものを二匹ずつ採っていく、二角取り（四川省）パズルです。
同じ絵柄の牌を 2 枚えらぶと、3 回以内の折れ線でつながる場合に消えます。
盤面をぜんぶ採りきればクリア。タコのテーマと、本格的な麻雀牌テーマを切り替えて遊べます。

▶ **[ブラウザですぐ遊ぶ](https://nishiokya.github.io/takodori/)**

---

## 遊び方

1. 同じ絵柄の牌を 2 枚タップ（クリック）します。
2. 2 枚のあいだが **2 回まで曲がる線**（外周をぐるっと回ってもよい）でつながれば、その 2 枚が消えます。
3. つながらない組み合わせは消えません。別の牌をえらび直してください。
4. 盤面の牌をすべて消すとクリア。これ以上消せる組み合わせがなくなると「手づまり（ゲームオーバー）」です。

### ルールのポイント

- つなぎ線は **タイルのない空きマス**と**盤の外周**だけを通れます。他の牌の上は通れません。
- 曲がり角は最大 2 か所まで（＝直線・L 字・コの字までつながる）。
- かならず解ける盤面だけが配られます。

## 操作・機能

| 機能 | 説明 |
| --- | --- |
| **ヒント** | 消せるペアを 1 組ハイライトします（スコアが半分になります）。 |
| **新しいゲーム** | 盤面をシャッフルして最初からやり直します。 |
| **テーマ切替** | 「タコ」🐙 と「麻雀」🀄 を切り替えます。 |
| **🕓 履歴** | 過去に遊んだ盤面（クリア／手づまり）をスコア付きで一覧し、同じ盤面に再挑戦できます。 |
| **𝕏 でシェア** | いま遊んでいる盤面の URL 付きでシェアします。 |

## 中断からの再開

遊びかけの盤面は自動でブラウザに保存されます。**スマホでアプリやタブを閉じて再起動しても、続きから再開**できます（盤面・スコア・経過タイム）。クリアや手づまりになると保存はクリアされます。

## 盤面コードと URL 共有

各盤面には `k-xxxx` のような短いコードがあり、**テーマ・乱数シード**を含みます。盤サイズはテーマごとに固定です。
このコードを使えば、**まったく同じ盤面を誰とでも共有**できます。

- URL に載る: 遊んでいる盤面は `…/?b=<コード>` の形で URL に反映されます。**この URL を開くと同じ盤面から始められます**。
- 🕓 履歴パネルの「URL をコピー」で共有 URL を取得。コードや URL を貼り付けて「この盤面で遊ぶ」でも再現できます。
- テーマだけ指定したいときは `…/?theme=mahjong`（または `?theme=tako`）で開くと、そのテーマの新しい盤面が始まります。

## 記録

- **自己ベスト**：クリアタイムの自己ベストをテーマ・難易度ごとに保存します。
- **スコア**：ペアを消すと `100 × コンボ数` 点が加算されます。
- **コンボ**：前のペアを消してから 5 秒以内に次のペアを消すと、コンボ数が 1 ずつ増えます。5 秒を超えると 1 コンボに戻ります。
- **ヒントのペナルティ**：ヒントを使うたびに、その時点のスコアが半分になります（小数点以下は切り捨て）。
- スコア・ベスト・履歴・遊びかけの盤面はすべてブラウザの `localStorage` に保存され、サーバーには送信されません。

## テーマ

| テーマ | 種類 | 盤面（縦×横） | 枚数 |
| --- | --- | --- | --- |
| **タコ 🐙** | 海の生きもの20種類 | 8×15＝120枚 | 各種類6枚 |
| **麻雀 🀄** | 通常牌34種類＋花牌4種類＝38種類 | 8×18＝144枚 | 通常牌は各4枚、花牌は各2枚 |

盤面はPCなどの横長画面では横15×縦8（麻雀は横18×縦8）、スマホ縦持ちでは横8×縦15（麻雀は横8×縦18）で表示します。

- **タコ 🐙** — タコ・イカ・カニ・クラゲ・ウミガメなど20種類の海の生きものです。
- **麻雀 🀄** — 萬子・筒子・索子・字牌をすべて手描きの図柄で表現。一索はテーマに合わせてタコになっています。
